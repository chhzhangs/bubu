
#创业者交流平台

##项目描述
使用ApiCloud和 Node.js 完成一个创业者匹配交流平台，用户能够通过该平台搜索到自己位置附近的并且有特定技能（前端、后台、设计、有钱任性等）的创业者，从而方便创业者们的交流和合作。同时，该平台提供一些活动的发布。
